# TiendaCTRLgamer
Es proyecto final de ecommerce basico para condehose utilizando React, El sitio te permite ver productos  y realizar compras en linea, los productos se alamacenan en Firebase


## Tecnologis utlizadas

- Vite
- React
- React-Router-DOM
- Toastify
- Firebase



## Funciones
- ver una lista de productos disponibles por stock
- Agregar productos de carrito de compras
- Ver carrito de compras
- Realizar una orden de compra con los datos del usuario
